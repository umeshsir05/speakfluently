🎯 Present Perfect Continuous Tense

👉 इसका उपयोग तब होता है जब कोई काम Past में शुरू हुआ था और अभी भी जारी है, या उसका असर अभी तक है।
(Used to show an action that started in the past and is still continuing OR its effect is still visible in the present.)

1. Affirmative (Positive) Sentence

Structure: Subject + has/have + been + V1 + ing + Object + since/for + time.

I have been playing cricket for two hours. (मैं दो घंटे से क्रिकेट खेल रहा हूँ।)
She has been reading a book since morning. (वह सुबह से किताब पढ़ रही है।)
2. Negative Sentence

Structure: Subject + has/have + not + been + V1 + ing + Object + since/for + time.

I have not been watching TV since morning. (मैं सुबह से टीवी नहीं देख रहा हूँ।)
He has not been working here for five years. (वह पाँच साल से यहाँ काम नहीं कर रहा है।)
3. Interrogative Sentence (Yes/No Questions)

Structure: Has/Have + Subject + been + V1 + ing + Object + since/for + time ?

Have you been studying since morning? (क्या तुम सुबह से पढ़ाई कर रहे हो?)
Has she been waiting for you for two hours? (क्या वह तुम्हारा दो घंटे से इंतज़ार कर रही है?)
4. Wh-Interrogative Sentence (Question word + Interrogative)

Structure: Wh-word + has/have + Subject + been + V1 + ing + Object + since/for + time ?

How long have you been living here? (तुम यहाँ कब से रह रहे हो?)
What has he been doing since morning? (वह सुबह से क्या कर रहा है?)
⏰ Since vs For

Since = जब exact point of time दिया हो (morning, 2010, Monday, yesterday). (e.g., since morning, since 2015)
For = जब duration/period दिया हो (two hours, five years, a long time). (e.g., for two hours, for five years)

🧠 Quick Trick

“Has been / Have been” → बताता है कि काम Past से शुरू हुआ और Present तक जारी है।
Verb + ing → काम जारी है।
Since/For → समय की जानकारी देता है।

✅ Example Recap:
I have been studying since 7 o’clock. (मैं 7 बजे से पढ़ रहा हूँ।)
They have been playing for three hours. (वे तीन घंटे से खेल रहे हैं।)
