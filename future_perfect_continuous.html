Future Perfect Continuous Tense

🌟 Future Perfect Continuous Tense (भविष्य पूर्ण अपूर्ण काल)

🔹 Definition (परिभाषा)

It shows an action that will have been continuing for a period of time at a certain point in the future.
यह उस क्रिया को दर्शाता है जो भविष्य में किसी समय तक लगातार चलती रहेगी।

👉 Keywords: for, since, by the time, until, for two hours, since morning, etc.
👉 मुख्य शब्द: से, तक, तब तक, घंटों से, सुबह से

✨ Formula (संरचना)
1. Affirmative (सकारात्मक वाक्य)

Structure: Subject + will have been + V1 + ing + object + since/for + time

I will have been studying for two hours by 8 pm.
(मैं रात 8 बजे तक दो घंटे से पढ़ रहा होऊँगा।)
2. Negative (नकारात्मक वाक्य)

Structure: Subject + will not (won’t) have been + V1 + ing + object + since/for + time

She will not have been working here for five years by next month.
(वह अगले महीने तक यहाँ पाँच साल से काम नहीं कर रही होगी।)
3. Interrogative (प्रश्नवाचक वाक्य)

Structure: Will + subject + have been + V1 + ing + object + since/for + time?

Will you have been waiting for me for an hour by 10 o’clock?
(क्या तुम 10 बजे तक मेरा एक घंटे से इंतज़ार कर रहे होगे?)
4. Negative Interrogative (नकारात्मक प्रश्नवाचक वाक्य)

Structure: Will + subject + not (won’t) + have been + V1 + ing + object + since/for + time?

Won’t they have been playing cricket for two hours by evening?
(क्या वे शाम तक दो घंटे से क्रिकेट नहीं खेल रहे होंगे?)
📝 Quick Recap (संक्षेप में)
Affirmative: Subject + will have been + V1 + ing + object + time
Negative: Subject + will not have been + V1 + ing + object + time
Interrogative: Will + subject + have been + V1 + ing + object + time?
Neg. Interrogative: Will + subject + not have been + V1 + ing + object + time?
